# Chtholly Language Development Roadmap

This document outlines the development plan for the Chtholly programming language.

## Phase 1: Core Language Features

- [x] Comments
- [x] Main function
- [x] Immutable and mutable variables
- [x] Data types
- [ ] Type casting
- [x] Operators
- [x] Functions
- [ ] Classes and Structs
- [ ] Control flow (if/else, switch)
- [ ] Loops (while, for, do-while)

## Phase 2: Advanced Features

- [ ] Generics
- [ ] Modules and imports
- [ ] Error handling (Result<T, E>)
- [ ] Compile-time programming (const system)
- [ ] Standard library

## Phase 3: Tooling and Ecosystem

- [ ] Package manager
- [ ] Formatter
- [ ] Language server
